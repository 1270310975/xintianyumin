# kldns
快乐二级域名分发系统

##此系统有哪些特点
* 目前支持的域名解析平台有
    *  dnspod
    *  cloudxns
    *  aliyun
* 多用户、多域名、多平台同时存在
* 界面简单、舒适，操作简单

##1.0->1.1 更新内容
* 添加新纪录时设置验证码，以防有人刷记录
* 后台密码可以修改了，在网站设置里面！
* 新增加域名可以指定只允许某一个用户使用！
